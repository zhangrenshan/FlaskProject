你好，我是{{name}}
        

        

            
jinja2变量索引

            我是name的索引位第一位：{{name.1}}
            

            
jinja2变量方法

            我是name的方法大写后：{{name.upper()}}
            

            
jinja2变量方法传参

            我是name的方法以a分割后：{{name.split('a')}}
            注意：参数需要加引号
            

            
jinja2循环

            {% for n in name %}
                我是name循环后的：{{n}}
            {% endfor %}
            

            
jinja2变量运算

            我是name加字符串后：{{name+'好帅'}}
            

            
jinja2判断

            {% if age %}
                如果有name，打印：
{{name}}

            {% else %}
                如果没有name，打印：
没有名称

            {% endif %}
